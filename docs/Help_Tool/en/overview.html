Overview

Operating environment, necessary runtime etc.

Editing tool

  • PC with Windows Vista or later equipped with ShaderModel 2.0 compatible video card
  • Microsoft .NET Framework 4(Full)
  • The latest DirectX end user runtime

Runtime (official)

  • DirectX 9 (ShaderModel 2.0 or later)
  • OpenGL 4.1 or later
  • OpenGL ES 2.0 or later

Installation, uninstallation

The editing tool exists in "Tool/". We do not operate the registry etc at all. Please start directly or copy to any directory.
When uninstalling, please delete each directory.

When playing effects in the game, please download separate packages such as runtime, Unity, DX libraries.

If you want to use the effect is output to the image, such as a serial number image, Recording references please.
