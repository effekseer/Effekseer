Attraction Forces

Overview

ノードの位置に関するパラメーターの中でも、特に親パーティクルに影響されない位置に関するパラメーターを設定します。

パラメーター

なし

設定しません。

重力

ある方向に加速度をかけます。下とは限らず、上や右にも適用することができます。

引力

パーティクルが「振る舞い」の「ターゲット位置」に指定した位置に向かうように移動します。

引力

1フレームにターゲット位置に向けて加速する力です。

制御

1フレームにおいて移動方向をターゲット方向に補正する割合です。0.00～1.00の値をとります。

最小範囲

引力の減衰開始距離です。

最大範囲

引力の減衰終了距離です。 パーティクルの位置が0～最少範囲の場合、引力100%になり、最大範囲～の場合、引力0%になり、最小範囲～最大範囲の場合、線形に減衰します。
